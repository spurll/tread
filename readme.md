TRead
=====

A very simple terminal RSS feed reader written in Python 3. It is probably not as good as the others that already exist.

Installation
============

Instructions
------------

Add the desired feeds to `config.yml` and run `python3 tread.py`.

Requirements
------------

* pyyaml
* requests
* beautifulsoup4
* lynx or w3m
